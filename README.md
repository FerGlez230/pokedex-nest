# Execute in development
1. Clone repository
2. Execute
```
npm install
```
3. Have Nest Cli installed
```
npm i -g @nestjs/cli
```
4. Load database
```
docker compose up -d
```
5. Execute seed to bring data to db
```
http://localhost:3000/api/v2/seed
```

## Stack usado
* MongoDB
* Nest
